# 本地调试指南

## 启动服务器

在项目根目录下打开终端，运行：

```powershell
$env:PATH = "D:\Ruby34-x64\bin;" + $env:PATH; bundle exec jekyll serve
```

启动后浏览器访问 http://127.0.0.1:4000

## 重启服务器

1. 在运行服务器的终端里按 `Ctrl + C` 停止
2. 重新运行上面的启动命令

> 修改 `_config.yml` 后必须重启服务器才能生效，修改其他文件（如 `_pages/about.md`）会自动刷新。

## 换端口启动

如果 4000 端口被占用：

```powershell
$env:PATH = "D:\Ruby34-x64\bin;" + $env:PATH; bundle exec jekyll serve --port 4001
```

## 安装依赖（首次或 Gemfile 变动后）

```powershell
$env:PATH = "D:\Ruby34-x64\bin;" + $env:PATH; bundle install
```
